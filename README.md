# MesureSurGrandGraphe
Associés au Big Data, les grands. Graphes (réseaux sociaux, infrastructure, biologie, etc) font. L’objet d’une attention grandissant dans différents domaines. Si un graphe ne possède pas de circuit(c’est à dire est une forêt ou un arbre), alors de nombreux calculs sur celui-ci deviennent plus faciles et rapides à opérer. De nombreuses façons de mesurer de combien un graphe “s’éloigne” d’un ambrer ont été proposées, parmi elles l’hyperbolicité et la dégénérescence sont les deux mesures les plus courantes et pour lesquelles on dispose d’algorithmes performant pour les calculer.  Ainsi, l’objectif de ce projet est d’implanter les algorithmes de calcul d’hyperbolicité te. De dégénérescence pour les tester sur des graphes réels (potentiellement assez grands) et. Ainsi vérifier en quelles mesures les graphes réels sont proches des arbres. Programmé en java.
